JavaScript

JavaScript (English: JavaScript) is an object-based script programming language.
It is also used in server side network programming, such as runtime environments such as Node.js.
JavaScript was originally developed by Netscape Communications Corporation's Brendan Eich under the name Mocha and later by LiveScript, which eventually became JavaScript.
Although Javascript's syntax is similar to that of Sun Microsystems' Java, this is in fact because both languages are based on the basic syntax of C language, and Java and JavaScript are not directly related.

As of January 2013, the latest version is JavaScript 1.8.5 and is supported on [3], Firefox 3.
The JavaScript version corresponding to the standard ECMA-262 3rd edition is 1.5.
ECMA Script is a standardized version of JavaScript, to put it simply.
